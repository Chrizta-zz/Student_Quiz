 🎯 Student Quiz using Hand Gesture Recognition

An interactive AI-powered quiz application that allows students to answer multiple-choice questions using **hand gestures** instead of a keyboard or mouse. The project uses **MediaPipe Hand Landmarker** and **OpenCV** for real-time hand tracking and gesture recognition.



 📌 Features

* 🖐️ Real-time hand gesture detection
* 🎥 Webcam-based interaction
* 📖 Multiple Choice Questions (MCQs)
* ✅ Select answers using finger gestures
* 📊 Automatic score calculation
* ⚡ Smooth and user-friendly interface
* 🤖 AI-powered computer vision using MediaPipe



 🛠️ Technologies Used

* Python 3.11
* OpenCV
* MediaPipe
* NumPy

🎮 How It Works

1. Launch the application.
2. The webcam starts automatically.
3. Hand gestures are detected in real time.
4. Point to the desired answer option using your finger.
5. Hold the gesture briefly to confirm your selection.
6. The application records the answer and moves to the next question.
7. At the end of the quiz, your final score is displayed.
